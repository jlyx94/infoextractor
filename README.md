<h1>Info Extractor</h1>

<h2>Description</h2>
Bash script that digs into the system to pull out key details, including IP and MAC addresses. A report is also included to explain the importance of knowing the extracted key details.
<br />


<h2>Languages Used</h2>

- <b>Bash</b>

<h2>Environments Used </h2>

- <b>Linux</b>
